# CV (Next.js) desde tu HTML — Listo para subir

## Cómo correr
1) Instala Node.js 18+.
2) Abre esta carpeta en VS Code.
3) Copia `.env.example` → **`.env.local`** (puedes dejar valores por defecto).
4) En el menú **Terminal** de VS Code:
```
npm install
npm run dev
```
5) Abre http://localhost:3000

## Editar tu contenido
- Tu HTML está en: `public/static/content.html` (ya pegado).
- Tu CSS de estilos está en: `styles/royal-sapphire.css` (migrado desde `<style>`).
- Tu foto va en: `public/assets/foto.jpg` (hay un placeholder).
